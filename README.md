# How to use

- Have <a href="https://www.python.org/">Python</a> installed.

- First clone the repository or <a href="https://github.com/efekrbas/e-z.bio-username-checker/archive/refs/heads/main.zip">download it as .zip</a>

- Install the required libraries: open `install.bat`

- open `ezbio.py`
